## Data Extraction

Data is extracted from MySQL using SQLAlchemy.

```python
from sqlalchemy import create_engine
import pandas as pd

engine = create_engine(
    "mysql+pymysql://root:@localhost/olist_project"
)

query = """
SELECT
    c.customer_unique_id,
    o.order_id,
    o.order_purchase_timestamp,
    oi.price
FROM customers c
JOIN orders o
    ON c.customer_id = o.customer_id
JOIN order_items oi
    ON o.order_id = oi.order_id
"""

df = pd.read_sql(query, engine)
```

---

## Customer Revenue Calculation

Total revenue generated by each customer is calculated using:

```python
customer_revenue = (
    df.groupby("customer_unique_id")["price"]
      .sum()
      .reset_index()
)

customer_revenue.rename(
    columns={"price":"total_spent"},
    inplace=True
)
```

---

## Customer Segmentation

Customers are segmented based on their spending behavior.

| Total Spend | Segment |
|------------|----------|
| >= 154 | High Value |
| >= 90 | Medium Value |
| < 90 | Low Value |

```python
def customer_segment(amount):

    if amount >= 154:
        return "High Value"

    elif amount >= 90:
        return "Medium Value"

    else:
        return "Low Value"
```

---

## Master Dataset Creation

The customer segment information is merged back into the transaction dataset.

```python
master_df = pd.merge(
    df,
    customer_revenue[
        ["customer_unique_id","segment"]
    ],
    on="customer_unique_id",
    how="left"
)

master_df.to_csv(
    "master_sales_dataset.csv",
    index=False
)
```
